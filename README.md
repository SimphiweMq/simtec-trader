# Simtek Trader

Simteknologies trading signal engine — Phase 1. JSE equities and ZAR forex backtester and signal dashboard.

## Project Structure

- **src/** — Core trading modules (data fetcher, signal engine, backtester, utilities)
- **data/** — Market data and analysis outputs
- **notebooks/** — Exploratory analysis and research
- **tests/** — Unit tests

## Requirements

See `requirements.txt` for dependencies.

## Getting Started

1. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
